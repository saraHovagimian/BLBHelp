 Emisión

 Para tener en cuenta:

El tipo de liquidación seleccionado, debe tener activado el check box Actualiza Visor de 4ta.

Para visualizar correctamente la informacion es necesario configurar las variables y/o acumuladores que se desean ver al emitir el informe en el Configurador Visor de 4ta.categoria.

Esta información se guardara en el sistema a partir de las liquidaciones en que se active el visor de 4ta.
