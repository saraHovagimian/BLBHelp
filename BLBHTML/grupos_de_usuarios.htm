Grupos de usuarios

Mediante esta tabla se incorporan y habilitan los grupos de usuarios del sistema.

 Grupo

Se debe indicar el código del grupo (hasta 32 caracteres alfanuméricos)

 Descripción

Se puede indicar la descripción del grupo.

 Usuarios

 Usuarios

En la columna de la izquierda rotulada UserName se indica el código del usuario a incluir en el grupo (Sólo se exhiben los usuarios que pertenecen a las empresas a las que pertenece el grupo).

Una vez seleccionado el código del usuario, la columna Nombre se completa, automáticamente, con el nombre del mismo.

Para eliminar un usuario se debe seleccionar la línea correspondiente al usuario y presionar la tecla Suprimir.

 Empresas

Al incorporar un nuevo grupo al sistema, por defecto, éste queda habilitado para operar todas las empresas definidas en la base de datos. Esta ficha permite indicar en que empresas de la base de datos el grupo puede operar.

 Empresas

La tabla exhibe una línea por empresa.

En la columna Código se exhibe el código de la empresa en la base de datos y en la columna Nombre el nombre de la misma.

 Habilita op.

Activado indica que el grupo puede operar en la empresa.

 Invertir selección

Presionando este botón se cambia el estado de los campos, al estado contrario al que poseen.

 Definir seguridad

Este botón se habilita cuando el usuario pertenece al grupo Administradores y luego de indicar el nombre del grupo.

 Para tener en cuenta

Para acceder a este punto el usuario debe pertenecer al grupo Administradores.

En cada grupo, únicamente pueden incluirse usuarios que pertenezcan a la empresa a la que pertenece el grupo.

El grupo Todos está precargado en el sistema e incluye a todos los usuarios existentes. De este grupo no puede excluirse ningún usuario. El grupo Todos tiene, inicialmente, todos las acciones permitidas.

El grupo Administradores está precargado en el sistema. Este grupo tiene siempre, sin excepción, todas las acciones permitidas, por lo tanto no pueden definirse permisos para el mismo. Inicialmente, el único usuario incluido en este grupo es sa. Este usuario no puede excluirse de este grupo.

El grupo Administradores pertenece a todas las empresas y no es posible excluirlo de ninguna de ellas.

Es importante tener en cuenta a que empresas pertenece cada grupo ya que la definición de permisos se realiza por grupo por empresa, es decir, si un grupo pertenece a más de una empresa y se desea definirle permisos en todas las empresas, la definición de permisos se debe realizar tantas veces como sea necesario, ingresando, en cada caso, a cada empresa.
