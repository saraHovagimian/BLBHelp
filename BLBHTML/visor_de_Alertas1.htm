Visor de alertas

Desde aquí es posible visualizar las alertas de Liquidación si en el configurador de alertas se indicó que las mismas se avisen por Agenda.

Borrar

Este botón se habilita cuando se selecciona una alerta de la lista de la derecha y sirve para borrar una o varias alertas.

Actualizar servicio

Este botón debe presionarse si se desea actualizar la información del  tipo de alerta que se encuentra seleccionado. Si no se indica un tipo en particular, se actualizan todas las alertas. Este botón sólo se habilita cuando el “Servicio de alertas” está iniciado.

Salir

Presionar este botón para salir del visor.

 Tipos de alerta

Se visualizan la lista de alertas ordenadas por Tipo.

 Lista de alertas

Lista de las alertas para el tipo seleccionado.

 Barra de estado

Se exhibe la siguiente información: Nombre del equipo donde está instalado el servicio de alertas,estados del servicio, Fecha y hora de la última lectura del servicio, Tiempo restante (expresado en minutos) para la próxima lectura y si el servicio está recolectando información.

 Para tener en cuenta:

Para que la agenda de alertas pueda visualizarse se debe tener instalado e iniciado el Servicio de alertas.
