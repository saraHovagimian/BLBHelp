Evaluación

A través de esta funcionalidad es posible gestionar y registrar diferentes tipos de evaluaciones. Las mismas pueden ser, desde la evaluación de desempeño periódica realizada por la organización hasta una simple evaluación de un curso generada en forma eventual.

 Código

Campo alfanumérico identificatorio de la evaluación a realizar.

 Descripción

Detalle de la evaluación a generar que posibilita brindar una idea principal de los puntos a evaluar.

 Escala

Código de la escala a utilizar, se asignará una calificación de la evaluación tomando como base los valores detallados en dicha tabla.

Para cada evaluación se debe definir una escala. Esta será la misma para todo el conjunto de formularios que componen la evaluación. En caso que la evaluación tenga algún formulario asociado, no se podrá modificar la escala.

Las escalas se dan de alta desde el menú principal.

 Tiene vencimiento

Permite identificar si la evaluación tiene una fecha de caducidad.

Sólo para  novedades explícitas

Si se activa este campo es posible asociar a la evaluación en cuestión (evidencia) a una o varias habilidades puntuales. Dichas habilidades se podrán seleccionar en la solapa Habilidades asociadas. .

 Formulario

Formularios

Código y descripción del formulario relacionado.

 Capacitaciones sugeridas

 Capacitaciones sugeridas

Son aquellas Capacitaciones que se establecen como necesarias para cumplimentar con la evaluación.

Pueden ser condiciones necesarias pero no obligatorias.

 Habilidades asociadas

Habilidades

Se identificarán las habilidades asociadas a la evaluación.

 Para tener en cuenta:

Si se activa el campo Sólo para novedades explícitas, al dar de alta una Novedad de evaluación de dicha evaluación, será necesario especificar para qué habilidad o habilidades en particular se está dando de alta esa evidencia.

Este tipo de evaluaciones sirve para poder definir evidencias genéricas, que se utilicen para evidenciar diferentes habilidades y para poder especificar claramente a cuál de las habilidades se está evidenciando al momento de dar de alta la novedad.

Cuando no se tilda la opción Solo para novedades explicitas, al dar de Alta una novedad de evaluación todas las habilidades que tengan esa evaluación como evidencia se van a dar por cumplidas.
