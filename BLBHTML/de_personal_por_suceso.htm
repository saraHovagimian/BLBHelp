De personal por suceso

Esta tarea permite informar, novedades de personal en relación a un suceso para un, o más, legajos.

Las novedades de personal también pueden ingresarse mediante las tareas destinadas a registrar novedades de liquidación Por legajo, Por concepto o novedades de personal Por legajo.

 Suceso

Según el tipo de novedad a informar la Clase de sucesos asociada al Suceso debe tener asociado alguno de los siguientes tratamientos:

Contrato de trabajo

Ingreso

Reingreso

Egreso

Modificación de fechas

Evaluación

Sanción disciplinaria

Adelanto de haberes

Sin particularidad

Los Sucesos deben haberse definido como No vinculados a la liquidación.

Se debe tener en cuenta que sólo se pueden informar novedades para los sucesos no denegados para el usuario actual según la definición de permisos.

 Fecha de ocurrencia

Fecha en que se registra la novedad.

 Legajos

Permite registrar los legajos para los cuáles se informa la novedad y los detalles concernientes a la misma.

Se debe tener en cuenta que sólo se pueden informar los legajos no denegados para el usuario actual según la definición de permisos.

 Novedades

La información que solicita este cuadro varía según el tratamiento correspondiente al Suceso que se ha informado.

 Legajos

Código del legajo para el cual se informa la novedad.

El código puede digitarse y también, en caso que la novedad se informe para varios legajos, es posible seleccionarlos conjuntamente en la ventana de búsqueda y selección.

 Nombre

Exhibe el nombre del legajo.

 FecIngreso

En caso de sucesos de tipo Ingreso o Reingreso, se debe informar la fecha de ingreso del legajo.

 FecBaseAnt

En caso de sucesos de tipo Ingreso o Reingreso, se debe informar la fecha a partir de la cual se comienza a computar la antigüedad.

La fecha base antigüedad también puede modificarse directamente desde la tabla Legajos.

 Días

En caso de sucesos de tipo Sanción disciplinaria, se debe informar la cantidad de días correspondientes.

 Adelanto

En caso de sucesos de tipo Adelanto haberes, se debe informar el importe del adelanto.

 FecPrevNotific

Fecha en que se debe presentar la notificación. Sólo disponible en caso que el suceso indique que se requiere notificación.

 Notificado

Tildar en caso que ya se cuente con al notificación. Sólo disponible en caso que el suceso indique que se requiere notificación.

 Comentarios

Si se requiere, texto aclaratorio de la notificación. Sólo disponible en caso que el suceso indique que se requiere notificación. El comentario que se informe está accesible desde la tarea Novedades de notificaciones.

 Fecha suspensión

Se habilita para sucesos que tienen asociado el Tratamiento Sanción disciplinaria.

Fecha a partir de la cual se hace efectiva la sanción.

 Venc. contrato

Se habilita para sucesos que tienen asociado el Tratamiento Contrato de trabajo.

Fecha de vencimiento del contrato de trabajo.

 Grabar

Si las variables o sucesos establecen topes y éstos se han superado, se emite un mensaje de advertencia o prohibición, según corresponda.

Si la novedad tiene un documento asociado se pide confirmación para continuar con la tarea y, en caso afirmativo, se da la posibilidad de imprimir, o no, dicho documento.

 Importación de datos desde archivos externos

 Para tener en cuenta:

La novedad se considera como una unidad, es decir que, posteriormente, no se permitirá efectuar eliminaciones parciales de los legajos que se incluyen.
