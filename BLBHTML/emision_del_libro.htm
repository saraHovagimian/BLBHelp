Emisión del libro

Mediante esta tarea se emite el libro de Sueldos y Jornales.

Es posible realizar la emisión sobre hojas rubricadas previamente. Para ello se prevé la tarea Títulos para rubricación destinada a imprimir solamente los títulos del libro sobre hojas blancas para su posterior rubricación.

Se debe tener en cuenta que sólo puede imprimirse el libro para los legajos que no tengan denegado el permiso correspondiente en la tarea Ver datos de la liquidación.

 Rangos

Esta ficha está destinada a indicar que información se incluye en el informe, su título y otros datos que se requieren para realizar la emisión.

 Incluir

Permite indicar que datos se requieren en el informe.

 Familiares

Activando este campo se incluyen los familiares que se han informado, en relación a cada legajo, en la tabla Familiares.

Para cada uno de ellos se exhibe el nombre y apellido, el parentesco, la fecha desde la que está a cargo, el tipo y número de documento y el texto que figure en el campo Observaciones.

 Espacios para firmas

Incluye una línea por empleado destinada a que éste firme el informe.

 CUILes

Permite incluir el número de C.U.I.L. que corresponde al legajo.

 Nros. de documento

Permite incluir el tipo y número de documento que corresponde al legajo.

 Títulos

Si está activado el informe incluye los títulos.

En caso que la emisión se realice sobre hojas rubricadas, no debe activarse.

 Fecha de emisión

En caso que esté activado, el encabezado del informe incluye fecha y hora de emisión.

 Título

Texto que se imprime, centrado, en el encabezado del informe.

 Subtítulo

Texto que se imprime debajo del informado en Titulo.

 Liquidación

 Tipo

Tipo correspondiente a la liquidación a imprimir. En caso que se desee imprimir todas las liquidaciones del mes, no seleccionar liquidación en este campo dejando el mismo en blanco.

 Mes

Mes correspondiente a la liquidación a imprimir.

 Emisión

 Legajos

 Para tener en cuenta:

El libro de sueldos y jornales se imprime por liquidación.
