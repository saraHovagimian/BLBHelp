Emisión

 Formato personalizado de impresión

Nombre del archivo correspondiente al formato en el que se quiere realizar la emisión.

Los archivos de formato de impresión se diseñan haciendo uso del Crystal Reports.

Si no se ingresa información en este campo, la emisión se realiza utilizando el formato estándar del sistema.

 Destino

Permite seleccionar por donde se realiza la emisión.

 Pantalla

Seleccionando esta opción el informe se emite por pantalla.

 Impresora

Esta opción está destinada a imprimir el informe.

 Selección de impresora

Permite seleccionar la impresora por la cual se realiza la emisión y acceder al panel de configuración de cada una de ellas.

 Excel

Esta opción genera un archivo xml con la información correspondiente a la emisión solicitada. Además genera un archivo de formato con el mismo nombre seguido por la extensión _CFG, que puede ser modificado para dar formato a las columnas.

La ubicación de los archivos generados es fija: C:\Users\Public\BASLaboro\Excel\Nombre del Punto de Menú.

 Archivo destino

 Se indica el nombre del archivo. De no especificarse, toma por defecto el nombre del punto del menú.
