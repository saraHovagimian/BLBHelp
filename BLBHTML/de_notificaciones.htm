De notificaciones

Esta tarea está destinada a indicar que las notificaciones que se encuentran pendientes han sido cumplidas y la fecha correspondiente a dicho cumplimiento. También permite volver atrás una novedad de notificación, es decir, una vez notificada poder deshacer la acción y dejarla, nuevamente, pendiente.

 Rango de sucesos

 Suceso

Es posible indicar uno o más sucesos cuyas novedades asociadas se desea exhibir.

Si el campo se deja en blanco se considerarán todos los sucesos.

 Rango legajos

 Legajo

Es posible especificar los códigos de legajos a incluir en el listado.

En caso de incluir más de uno se deben separar con comas (,). También es posible indicar rangos utilizando guiones (-).

Se debe tener en cuenta que sólo se pueden informar los legajos no denegados para el usuario actual según la definición de permisos.

 Rango notificados

Permite seleccionar si se consultarán solo las novedades notificadas, las pendientes de notificación o todas.

 Todas

Exhibe todas las novedades que requieren notificación, ya sea que están notificadas o que no lo estén.

 Notificado

Exhibe solamente las novedades notificadas.

 No Notificado

Exhibe solamente las novedades no notificadas.

 Novedades

En la grilla se exhiben las novedades que cumplen con los filtros establecidos.

 Suceso

Exhibe el código del suceso correspondiente a la novedad y dispone de un enlace que permite acceder a los datos del mismo.

 Descripción

Exhibe la descripción del suceso anterior.

 Legajo

Código del legajo correspondiente a la novedad.

 Nombre

Nombre del legajo correspondiente a la novedad.

 Notificada

Activando este campo se indica que la notificación ha sido cumplida.

Si por algún motivo se requiere volver atrás la indicación de notificación cumplida, bastará con destildar el campo.

 Fecha Notificado

Este campo es obligatorio en caso de haber tildado el campo Notificada.

Al tildar el campo Notificada el sistema propone, por defecto, la fecha del día de hoy, pudiendo reemplazarse por la fecha que corresponda.

 Fecha prevista

Se exhibe la fecha informada en el campo del mismo nombre de la solapa Notificaciones al ingresar las novedades.

 Comentario

Al marcar una novedad como notificada es posible insertar un comentario de hasta 255 caracteres.

 Grabar

El proceso graba las novedades notificadas.

 Para tener en cuenta:

Para que una novedad "Requiera notificación" es necesario que el suceso mediante el cual se informa la novedad se haya definido con el campo Requiere notificación activado.
