Informe de tenencias de elementos de trabajo

Este informe permite visualizar las tenencias de elementos de trabajo a una determinada fecha. Es posible informarse sobre los elementos en poder de los empleados, que requieran devolución.

Además, se puede solicitar información de elementos faltantes de entrega según el puesto de trabajo que la persona ocupa.

Elementos de trabajo

Permite seleccionar los elementos de trabajo que se desean consultar. Si no se ingresa ninguno, se asumen todos.

Opciones

Tenencias al

Permite establecer la fecha a la que se desea conocer las tenencias.

Incluir elementos faltantes de acuerdo al puesto de trabajo

Si está activado, el sistema informa sobre los elementos obligatorios establecidos para los puestos de trabajo, que a la fecha aún no han sido entregados.

Solo elementos que requieran devolución

Si está activado, el sistema informa solamente los elementos en poder de los empleados, que requieran devolución.

Agrupamiento

Por persona

Emite el informe por persona, detallando los elementos que posee cada una.

Por elemento

Emite el informe por elemento, detallando las personas que poseen dicho elemento.

Personas

Desde este link es posible seleccionar las Personas de las que se quiere visualizar el informe.

Se pueden seleccionar por Atributo, Lugar de Trabajo, Puesto, etc.

 Atributos

Permite filtrar los elementos de trabajo por atributos.
