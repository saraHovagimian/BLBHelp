Abrir

Permite administrar las conexiones a las bases de datos correspondientes al sistema e ingresar a una de ellas, en particular.

 Base de datos

Seleccionar la base de datos a la que se desea acceder.

Si se trabaja con seguridad integrada se exhibe la base de datos "por defecto" que corresponde al usuario.

 Usuario

Usuario que accede al sistema.

Se exhibe aquel que inició la sesión en el equipo. Si se trabaja con seguridad integrada, no se puede modificar.

 Contraseña

Clave de acceso correspondiente al usuario.

Al digitarla se exhibe un asterisco por cada carácter que se ingresa.

Si se trabaja con seguridad integrada el ingreso es directo.

 Aceptar

Valida los datos registrados y, si son correctos, accede a la base de datos.

En caso contrario, exhibe un mensaje alusivo al inconveniente detectado.

 Cancelar

Sale de la tarea sin realizar ninguna acción.

Para tener en cuenta:

Una vez que el usuario accede a la base de datos puede operar en todas aquellas empresas para las cuáles fue habilitado.

Para mayor información acerca de la configuración de acceso a las distintas conexiones de base de datos remitirse a la Cartilla de Instalación del producto.
