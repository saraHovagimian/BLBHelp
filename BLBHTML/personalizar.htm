Personalizar

Cuando se incorpora un nuevo usuario a la base de datos, por defecto, se le asocia el menú general del sistema, es decir, el que corresponde al administrador del sistema (sa).

Esta tarea permite personalizar el menú de cada usuario de modo de excluir submenús o puntos del menú, agregar acceso a otras aplicaciones, incorporar nuevas carpetas de modo de agrupar u ordenar las tareas de forma diferente.

 Menú

En esta ficha se define el menú a asociar a cada usuario.

 Origen

Permite indicar en base a que menú se generará el menú del usuario.

 Menú base

Al seleccionar esta opción se exhibe el menú standard de un usuario administrador.

 Otro usuario

Al seleccionar esta opción, se habilita el campo para seleccionar el usuario cuyo menú desea utilizarse como base.

 Destino

Permite configurar el menú del usuario.

 Seleccione un usuario

Usuario para el cual se personaliza el menú.

 Menú

Este cuadro exhibe el menú actual del usuario y permite su configuración según corresponda.

Si aun no se ha personalizado el menú para el usuario en cuestión, y éste está usando el menú de "sa", no se exhibe ningún menú.

Para incorporar un submenú, o punto de menú, al menú del usuario "Destino", se lo debe arrastrar, desde el "usuario origen", hasta el nivel del cual debe depender. También es posible utilizar las opciones del "Menú contextual" que se presenta al presionar el botón derecho del mouse.

 Menú contextual

Este menú se presenta al presionar el botón derecho del mouse sobre un punto de menú o sobre un submenú.

Agregar

Esta opción está disponible cuando se presiona el botón derecho del mouse sobre un submenú y permite incorporar, dentro del mismo, una nueva carpeta, un programa de BAS Laboro o un programa externo.

Si se selecciona "Programa" se abre una ventana de búsqueda donde es posible seleccionar el programa a incorporar al menú.

Si se selecciona "Programa externo" se abre un cuadro donde es posible indicar la aplicación a incorporar al menú.

Eliminar

Esta opción está disponible cuando se presiona el botón derecho del mouse sobre un punto de menú o submenú y permite su eliminación.

Cambiar nombre

Esta opción está disponible cuando se presiona el botón derecho del mouse sobre un punto de menú o submenú y permite modificar el nombre con el cual se exhibe.

 Flecha arriba

Mueve hacia arriba, manteniéndola en el mismo nivel, una opción de submenú o de punto de menú.

  Flecha abajo

Mueve hacia abajo, manteniéndola en el mismo nivel, una opción de submenú o de punto de menú.

 Para tener en cuenta:

Si en el campo "Destino" se selecciona un usuario para el cual aun no se ha efectuado la personalización, el sistema no exhibe ningún menú a pesar de que a dicho usuario le corresponde el menú correspondiente al usuario "sa". Pero, al momento que se informe un menú o punto de menú, ése será el menú que se le ofrezca al ingresar al sistema. En otras palabras, mientras no se indica nada para el menú de un usuario éste pueda utilizar el menú general del sistema pero, bastará con que se informe "algo", para que su menú incluya solamente los puntos que se han informado.

Al presionar el botón derecho del mouse sobre el menú principal del sistema no están disponibles las opciones "Eliminar" y "Cambiar nombre". Esto se debe a que este menú no puede ni modificarse ni eliminarse por ser el nivel máximo del sistema, es decir, todos los puntos que se incorporen, obligatoriamente, serán subniveles del mismo.
