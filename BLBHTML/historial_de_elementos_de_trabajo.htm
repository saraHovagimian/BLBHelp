Historial de elementos de trabajo

Este informe permite visualizar el histórico de entregas, devoluciones, roturas, pérdidas, reposiciones, etc de todos los elementos de trabajo.

Elementos de trabajo

Permite seleccionar los elementos de trabajo que se desean consultar. Si no se ingresa ninguno, se asumen todos.

Fechas

Permite establecer un rango de fechas a consultar para el informe. El sistema solo tiene en cuenta las novedades cargas durante el período solicitado.

Tipo de Novedad

Permite establecer qué tipo de novedades se incluyen o quitan del informe.

Por elemento
