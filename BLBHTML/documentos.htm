Documentos

Los tipos de documentos son evidencias físicas respaldatorias que acreditan una habilidad ó capacitación realizada.

 General

 Código

 Código que se asigna al tipo de documento, Ej.:(D001, TITULO, CV).

 Descripción

Nombre asignado al tipo de documento que se está definiendo (Ej. Título Universitario, Curriculum Vitae).

 Palabras claves

Puede asignarse una palabra ó código que permita realizar búsquedas sencillas de dichos documentos.

 Tiene vencimiento

Permite identificar si el documento tiene una fecha de caducidad, ó de renovación. Por ejemplo en los casos de renovación del registro de conductor o de matrículas.

Sólo para  novedades explícitas

Si se activa este campo es posible asociar el docuemnto en cuestión (evidencia) a una o varias habilidades puntuales. Dichas habilidades se podrán seleccionar en la solapa "Habilidades asociadas".

 Habilidades asociadas

Habilidades

Se identificarán las habilidades asociadas al documento.

 Para tener en cuenta:

Si se activa el campo Sólo para novedades explícitas, al dar de alta una Novedad de documento de dicho documento, será necesario especificar para qué habilidad o habilidades en particular se está dando de alta esa evidencia.

Este tipo de documentos sirve para poder definir evidencias genéricas, que se utilicen para evidenciar diferentes habilidades y para poder especificar claramente a cual de las habilidades se está evidenciando al momento de dar de alta la novedad.

Cuando no se tilda la opción Solo para novedades explicitas, al dar de Alta una novedad de documento todas las habilidades que tengan ese documento como evidencia se van a dar por cumplidas.
