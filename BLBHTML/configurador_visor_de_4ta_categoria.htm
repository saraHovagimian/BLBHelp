Configurador visor de 4ta categoría

Es posible indicar los códigos de las variables y/o de los acumuladores que se deseen visualizar en el visor de ganancias de 4ta categoría y de 4ta categoría anual.

 Variables del visor de 4ta categoría

En esta grilla aparecen las columnas:

Variable

Nombre de la variable (o acumulador) que se desea visualizar

Código

Se debe indicar la variable o el acumulador que tiene el valor que desea visualizarse

Descripción

Descripción de la variable o acumulador

 Modificar

Este botón permite modificar los registros de la grilla.

 Grabar

Para que los datos informados queden grabados en el sistema es necesario presionar este botón, si se cierra la ventana sin haber grabado, todos los valores se perderán.

 Cancelar

Limpia la pantalla de datos sin registrar los datos ingresados.

 Para tener en cuenta:

Si se desea guardar la informacion de 4ta. categoria, el tipo de liquidacion seleccionado debe tener activado el check box Actualiza visor de 4ta.
