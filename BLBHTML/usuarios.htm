Usuarios

Mediante esta tabla se incorporan y habilitan los usuarios del sistema. Estos deben haberse definido previamente como usuarios de la base de datos.

 Datos generales

Esta ficha agrupa los datos identificatorios del usuario y las empresas por defecto que le corresponden.

 Nombre

Este dato es de carácter informativo.

 E-mail

Dirección de e-mail del usuario.

 Empresa BAS Laboro

Este dato es de carácter obligatorio.

Indica la empresa a la que accede el usuario cuando ingresa al sistema.

 Empresa BASCS

Indica la empresa a la que accede el usuario cuando ingresa a BAS CS.

 Empresas

Al incorporar un nuevo usuario del sistema, por defecto, éste queda habilitado para operar y emitir informes en todas las empresas definidas en la base de datos. Esta ficha permite indicar en que empresas de la base de datos el usuario puede operar y, en cuáles, emitir informes.

 Empresas

La tabla exhibe una línea por empresa detallando las habilitaciones del mismo.

 Código

Código identificatorio de la empresa.

 Nombre

Nombre de la empresa.

 Habilita op.

Si está activado el usuario puede operar en la empresa.

Luego, las tareas a realizar en ella, pueden limitarse personalizando su menú y/o asignando permisos por programa.

 Habilita inf.

Si está activado el usuario puede emitir informes en la empresa.

Si se requiere, se puede limitar la emisión a determinados informes, personalizando su menú. También puede limitarse su accionar mediante configurando los permisos por programa según se requiera.

 Default inf.

Indica que la empresa correspondiente a la línea se incluirá, por defecto, en los informes multiempresa que emita el usuario.

 Invertir selección

Presionando este botón se cambia el estado de los campos Habilita operación, Habilita informes y Default informes, al estado contrario al que poseen.

 Informes multiempresa

Esta ficha permite definir el título por defecto que ofrece el sistema en los informes multiempresa.

 Descripción

Texto que se ofrece por defecto como título cada vez que se emite un informe multiempresa y que, en todos, los casos podrá modificarse libremente.

 Completar con empresas

Al presionar este botón se cargan, en el campo Descripción, los nombres de las empresas para las cuales, en la ficha Empresas, se indicó Habilita informes

 Grupos

Esta ficha sólo se habilita si el usuario del sistema pertenece al grupo Administradores.

Permite indicar en que grupo (o grupos) se encuentra cada usuario. Para incluir un usuario en un grupo basta con tildar el casillero correspondiente al mismo.

 Todos

Existe un grupo llamado Todos al que pertenecen todos los usuarios, con excepción del usuario sa.

 Definir seguridad

Este botón sólo se habilita cuando el usuario pertenece al grupo Administradores y luego de indicar el nombre del usuario.

 Para tener en cuenta:

Para acceder a este punto el usuario debe pertenecer al grupo Administradores.

Si el usuario debe operar en otra empresa puede acceder a ella mediante el submenú Cambio de empresa del menú Base de datos, sin necesidad de modificar la empresa por defecto que se le ha asociado.

Si por algún motivo a un usuario se le deshabilitan todas las empresas, se emite un mensaje informando que, al menos, debería tener una empresa habilitada y se vuelve al estado anterior al intento de modificación.

Es importante tener en cuenta a que empresas pertenece cada usuario ya que la definición de permisos se realiza por usuario por empresa, es decir, si un usuario pertenece a más de una empresa y se desea definirle permisos en todas las empresas, la definición de permisos se debe realizar tantas veces como sea necesario, ingresando, en cada caso, a cada empresa.
