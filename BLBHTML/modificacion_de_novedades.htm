Modificación de novedades

Permite modificar la vigencia de las novedades de liquidación, de personal y de rotación de personal. Para la modificación de novedades relativas a licencias y vacaciones se dispone de una tarea específica.

La tarea prevé también la posibilidad de suspender novedades informadas con anterioridad y, en caso que se requiera, eliminar suspensiones ya registradas, de modo que las novedades recuperen su vigencia original.

Con respecto a los permisos de seguridad, se debe tener en cuenta que sólo pueden modificarse aquellas novedades que el usuario actual pudo informar.

 Empresa

Exhibe la empresa actual, empresa para la cual se informa la novedad.

No se puede modificar.

 Novedad para:

 Según que la novedad sobre la que se requiera actuar se haya informado en relación a un legajo o a un atributo y valor del mismo, se deberá ingresar uno u otro.

 Novedad a modificar:

 Traer novedad

Al presionar el botón, se exhiben todas las novedades registradas en el sistema para el legajo o atributo y valor informados, de modo que se seleccione la novedad a modificar, a suspender o a eliminar la suspensión.

Si la novedad que se selecciona fue ingresada al sistema conjuntamente con otras, todas ellas se cargan simultáneamente en la tarea, de modo de afectarlas a todas a la vez. Al seleccionar la novedad se exhiben en pantalla la fecha de registración y el suceso al cual se han asociado.

 Vigencia

Este cuadro está destinado a la modificación de novedades.

Es posible modificar la fecha que fue informada en el campo "Hasta fecha" al momento de registrar la novedad; también es posible incorporar una fecha en dicho campo en novedades que fueron registradas como "Sin vencimiento". La única condición para poder efectuar la modificación es que no se hayan efectuado liquidaciones posteriores a la fecha de finalización de la novedad a modificar para el o los legajos involucrados.

 Desde fecha

Exhibe la fecha desde la cual está vigente la novedad y no se permite su modificación.

 Hasta fecha

Al informar una fecha el sistema controla que no se hayan efectuado liquidaciones posteriores a la fecha ingresada para el o los legajos involucrados.

En caso que se desee ingresar una fecha "Hasta" a una novedad que fue informada "Sin vencimiento", para poder acceder al campo, se debe destildar el campo "Sin vencimiento".

 Sin vencimiento

Si se activa permite indicar que la vigencia no tiene vencimiento.

Si se desactiva permite ingresar la fecha "Hasta fecha" correspondiente a la vigencia.

 No liquidar (Suspender la novedad)

Esta ventana está destinada a informar la suspensión de novedades o la eliminación de novedades registradas con anterioridad.

 Desde fecha

El campo puede exhibir dato, en caso que se traiga a la pantalla una novedad para la cual ya se ha informado una suspensión. En caso contrario se presenta sin dato.

Si se trae a la ventana una novedad vigente que se desea suspender, o se trae una novedad suspendida para la cual se desea modificar la fecha de suspensión, tanto en un caso como en otro, se debe informar la fecha desde la cual tendrá vigencia la suspensión.

Si lo que se desea es eliminar una suspensión ya registrada, no debe ingresarse información.

 Hasta fecha

El campo puede exhibir dato, en caso que se traiga a la pantalla una novedad para la cual ya se ha informado una suspensión. En caso contrario se presenta sin dato.

Si se trae a la ventana una novedad vigente que se desea suspender, o se trae una novedad suspendida para la cual se desea modificar la fecha de suspensión, tanto en un caso como en otro, se debe informar la fecha hasta la cual tendrá vigencia la suspensión.

Si lo que se desea es eliminar una suspensión ya registrada, no debe ingresarse información.

 Todas las liquidaciones

Este campo se debe activar en caso que la suspensión rija para todos los tipos de liquidaciones habilitados en el sistema..

 Tipo liquidación:

Si la suspensión sólo tendrá vigencia para un tipo determinado de liquidación se debe activar el campo e informar el tipo de liquidación del cual se trata.

Se debe indicar el tipo de liquidación deseado.

 Eliminar

Presionando este botón, previo pedido de confirmación, se elimina la suspensión registrada y la novedad recupera la vigencia que tenía antes de registrar dicha suspensión.

 Conceptos y Variables

Estos cuadros exhiben los conceptos y variables informados al sistema en relación a la novedad que se modifica.

 Para tener en cuenta

Al presionar "Aceptar", si existen liquidaciones posteriores a la fecha informada, se emite un aviso y no se permite continuar adelante con la tarea.

 Atención

En caso que se indique un legajo, se traiga la novedad a modificar a la pantalla y ésta informada conjuntamente para varios legajos, por ejemplo mediante la tarea de ingreso de novedades "Por concepto", al registrar la modificación se modificará la novedad para todos los legajos informados.

En esta situación, al presionar "Aceptar", el sistema emite un aviso advirtiendo que la modificación que se efectúe afectará a varios legajos de modo que se tenga una instancia para constatar si la modificación a efectuar corresponde o no.
