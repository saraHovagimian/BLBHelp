De liquidación

Este informe permite listar las novedades vinculadas a la liquidación que se han informado en relación a variables y conceptos.

Se debe tener en cuenta que sólo se exhiben en el informe los datos que en Definir seguridad

no tengan tildado Denegar para la tarea Ver novedades.

 Empresa

Exhibe la empresa actual, empresa para la cual se informa la novedad.

No se puede modificar.

 Rangos

Esta ficha permite establecer múltiples criterios para ordenar y filtrar la información que se desea incluir en el listado.

 Validez

 Período de vigencia

Corresponde a la fecha de vigencia de las novedades.

Si se selecciona esta opción el informe exhibe los conceptos y/o variables vigentes para el mes y el tipo de liquidación indicados.

 Mes

Mes y año correspondientes a las fechas de vigencia que se desean incluir en el informe.

 Tipo de liquidación

Tipo de liquidación que se desean incluir en el informe.

 Período de ocurrencia

Corresponde a la fecha de ocurrencia de las novedades.

Si se selecciona esta opción el informe exhibe los conceptos y/o variables por fecha de ocurrencia de las novedades para el rango de fechas indicado.

 Desde fecha-Hasta fecha

Rango de fechas correspondiente a las novedades a incluir en el informe.

 Ordenar por

 Legajo/Atributo/Convenio

Las novedades que incluye el informe (de acuerdo a la selección realizada en la solapa Legajos y atributos) se exhiben ordenadas por legajo, por atributo por convenio,.según se indique en este campo. Es decir que para cada uno de ellos se exhiben todas las novedades que se han informado en relación a los rangos establecidos.

 Concepto/Variable

Las novedades se agrupan por concepto o variable, según corresponda. Para cada concepto o variable del rango se exhiben todos los legajos para los cuales se han informado novedades.

 Mostrar

 Conceptos

Si se activa este campo, el informe incluye conceptos.

 Variables

Si se activa este campo, el informe incluye variables.

 Concepto

En caso que listar novedades en relación a un concepto se debe indicar la clase y el número correspondientes al mismo.

Si no se informa ningún concepto el informe incluye todos los conceptos informados como novedad que cumplen las condiciones establecidas

 Novedades Liquidar Si No/Susp.

Si sólo se indica Si, el informe incluye sólo aquellos conceptos que al momento de informarse por novedades se indica que se liquiden.

Si sólo se indica No/Susp., el informe incluye sólo aquellos conceptos que se informaron como novedades de suspensión.

Si se desean incluir en el informe todos los conceptos informados por novedades pueden tildarse ambos campos o pueden dejarse ambos campos sin tildar.

 Conceptos

Si se elige Todos, el informe exhibe todos los conceptos, incluidos los auxiliares, si se elige Sin auxiliares el informe sólo exhibe los conceptos que no son auxiliares y si se elige Sólo auxiliares el informe solamente exhibe los conceptos definidos como auxiliares.

 Variable

En caso que listar novedades en relación a una variable se debe indicar el código correspondiente a la misma.

Si no se informa ninguna variable, el informe incluye todas las variables informadas como novedad que cumplen las condiciones establecidas

 Emisión

 Legajos y atributos

 Filtro

 Legajo

Si se elige esta opción, el informe exhibe el legajo indicado o un rango de legajos mediante el uso de atributos

 Atributo

Si se elige esta opción, el informe exhibe las novedades informadas por el atributo y el valor del mismo seleccionados.

 Convenio

Si se elige esta opción, el informe exhibe las novedades informadas por el/ los convenio(s) de trabajo seleccionado(s).

 Listar sólo novedades dadas para el/los legajo/s

Este campo se habilita al seleccionar Legajo y se utiliza para indicar que el informe incluya sólo las novedades informadas para el legajo en forma directa, no por atributos.

 Para tener en cuenta:

La Consulta de legajos permite acceder rápidamente a las novedades de liquidación en relación a un legajo en particular.

La ficha Consulta de conceptos y variables vigentes del ingreso de Novedades por legajo también permite consultar las novedades relativas a un legajo en particular.
