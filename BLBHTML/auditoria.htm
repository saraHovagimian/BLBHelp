Consulta de Auditoría

Esta consulta permite establecer, todas aquellas acciones relacionadas con altas, bajas y modificaciones de archivos maestros, novedades y liquidaciones que pueda realizar un usuario o grupo de usuarios en un periodo determinado de tiempo

Para acceder a este punto el usuario debe pertenecer al grupo Administradores.

Filtro

Mediante este enlace se accede al filtro de selección que permite determinar las distintas opciones cuyo resultado se visualizara en la pantalla principal de la consulta.

Filtro de Selección

Usuario:

Permite seleccionar el nombre de usuario ó grupo de usuarios a ser auditado.

Validez

Desde: Fecha de inicio que se toma para obtener información de la consulta de auditoría.

Hasta: Fecha de finalización que se toma para obtener información de la consulta de auditoría.

Incluir liquidaciones y anulaciones

Permite establecer si en el resultado de la consulta se incluirán los registros correspondientes a liquidaciones y anulaciones realizadas por el usuario auditado.

Incluir Novedades

Permite establecer si en el resultado de la consulta se incluirán los registros de novedades asignados por el usuario auditado.

Solapa Novedades

En esta solapa se realiza la selección de campos deseados para obtener información de auditoría de novedades.

Suceso

Suceso

Permite seleccionar el código de suceso por el que se desea filtrar la información resultante.

Código Interno

Código

Permite seleccionar el código interno de la novedad a auditar determinado en la Administración de novedades.

Concepto

Clase

Clase de concepto que se desea auditar (Haberes, retenciones, Contribuciones, etc.)

Número

Número de concepto que se desea auditar.

Variable

Variable

Código de la variable que se desea auditar.

Novedades dadas por:

Legajo

Código de legajo por el cual se asignaron los registros de novedades a auditar.

Atributo

Código de atributo por el cual se asignaron los registros de novedades a auditar.

Valor

Código de valor de atributo por el cual se asignaron los registros de novedades a auditar.

Convenio

Código de convenio por el cual se asignaron los registros de novedades a auditar.

Incluir todas las novedades

Permite visualizar las novedades a auditar sin discriminar la forma en que fueron asignadas.

Ganancias

Sólo novedades importadas del SIRADIG

Si se desean visualizar únicamente las novedades importadas del SIRADIG, es necesario activar el check box.

 Rango de fechas

Los campos Desde y Hasta indican la vigencia de las novedades importadas del SIRADIG.

Solapa Maestros

En esta solapa se realiza la selección de campos deseados para obtener información de auditoría de archivos maestros.

Maestro

Código de archivo maestro que se desea asignar para realizar la auditoría.

Todos los ......

Si se selecciona esta opción y dependiendo del grupo de maestros seleccionado previamente se incluirán todos los registros que corresponden a dicho grupo (Ej. Todos los legajos, Todas las personas, Todos los conceptos, Todos los acumuladores, todas las variables).

Acumulador, Concepto, Legajo, Persona, Variable

Dependiendo del tipo de maestro seleccionado anteriormente, se permitirá seleccionar uno ó mas registros para la selección de la auditoría de maestros.

Incluir todos los campos

Si se selecciona esta opción y dependiendo del grupo de maestros seleccionado previamente se incluirán todos los campos que corresponden a las tablas de dicho grupo.

Campos

Dependiendo del tipo de maestros seleccionado, se podrá determinar la auditoría de los registros por un campo determinado. (Ej.: domicilio, CUIL, Centro de Costos, etc.)
