Autonumeración

Esta tarea está destinada a establecer metodologías de asignación automática de códigos a legajos y a personas.

En ambos casos los códigos pueden constar de hasta ocho caracteres y se componen de un Prefijo, un Número y un Sufijo. El prefijo y el sufijo pueden ser caracteres alfanuméricos y el número puede completarse con ceros a la izquierda.

El algoritmo asigna como código el que resulte de concatenar, prefijo, número y sufijo, completando o no con ceros el número, según se haya indicado.

 Legajos

Permite establecer criterios de codificación de legajos.

 Numerar automáticamente los legajos

En caso de estar activado, al dar un alta en la tabla Legajos se completa el campo Código en base a los datos suministrados en esta ventana.

El código asignado por el sistema puede modificarse libremente.

 Numeración

Establece el criterio de asignación de la parte numérica del código del legajo.

 Cantidad de dígitos

Corresponde a la cantidad de dígitos deseados para el número del código del legajo.

 Próximo número

Se puede indicar el número correspondiente al próximo legajo a dar de alta en el sistema.

En caso de tener tildado el campo Numerar automáticamente los legajos el número aquí indicado es el ofrecido al momento de dar de alta un nuevo legajo.

Este dato se actualiza automáticamente con cada alta de legajo.

 Completar con ceros

Si se tilda este campo, el sistema completa con ceros a la izquierda el número.

 Agregar

Además del número, es posible incorporar un prefijo y/o un sufijo al código de legajo.

 Prefijo

Se deben indicar los caracteres deseados para que aparezcan delante del número en el código del legajo.

 Sufijo

Se deben indicar los caracteres deseados para que aparezcan a continuación del número en el código del legajo.
