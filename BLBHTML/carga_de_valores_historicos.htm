Carga de valores registrables

En esta tarea se informan los valores para Atributos registrables -  Acumuladores registrables y Variables Registrables correspondientes a liquidaciones anteriores a la implementación de BAS Laboro en la empresa.

 Legajo

Código del legajo al cual se le ingresan los valores históricos de Atributos registrables, Acumuladores registrables y Variables registrables

Sólo pueden informarse los legajos habilitados de acuerdo a los permisos de seguridad definidos para el usuario.

 Ver todas las columnas

En este cuadro se tildan aquellos Atributos registrables, Acumuladores registrables y Variables registrables para los cuales se van a ingresar valores históricos.

A medida que se tildan los elementos disponibles, se habilitan columnas en la grilla inferior de la pantalla.

En caso de tildar el check box Ver todas las columnas automáticamente se visualizan en la grilla inferior todos los elementos disponibles para informar valores.

 Valores Registrados

En esta grilla aparecen como columnas fijas MesLiq y TipoLiq, mientras que el resto de las columnas aparecen sólo si fueron tildados los elementos en el cuadro Ver todas las columnas.

MesLiq

Mes correspondiente a la liquidación para el que se ingresan los valores correspondientes.

TipoLiq

Tipo de liquidación para el que se ingresan los valores correspondientes.

SUELDO

Columna que permite registrar valores históricos para el acumulador Sueldo, cargado como ejemplo de acumulador registrable.

CAR

Columna que permite registrar valores históricos para el Atributo registrable Car (Cargo), cargado como ejemplo de atributo registrable.

BASICO

Columna que permite registrar valores históricos para la Variable registrable BASICO, cargado como ejemplo de variable registrable.

 Imprimir

Permite realizar una impresión de los datos visualizados en la grilla.

 Importar

Permite levantar información desde un archivo de texto. Al presionar el botón se abre una ventana que permite importar registros desde un archivo plano csv (archivo de texto delimitado por comas o por el delimitador de listas definido en configuración regional del equipo).

El archivo debe contener una primera línea que indica el orden y el nombre de los campos que se importarán. Los nombres deben coincidir con los nombres de las columnas de la grilla.

Se requieren como columnas obligatorias: el tipo de operación (A si es un alta, B si es baja o M si es modificación), el legajo, el mes y año de liquidación y el tipo de liquidación y, luego, los acumuladores, atributos y variables que figuran en la grilla.

Los campos deben estar delimitados por ";" o por el carácter informado como delimitador, teniendo como campos obligatorios ABM ; LEGAJO ; MESLIQ ; TIPOLIQ, a continuación se indicaran los códigos de los acumuladores, los atributos y las variables con sus respectivas descripciones.

A partir de la segunda línea el archivo debe incluir los valores de los campos indicados en la primera línea, haciendo corresponder el orden de los mismos con dicha línea.

 Eliminar

Este botón permite borrar los registros seleccionados en la grilla. En caso de no seleccionar ninguno en particular, se eliminan todos.

 Grabar

Para que los datos informados queden grabados en el sistema es necesario presionar este botón, si se cierra la ventana sin haber grabado, todos los valores se perderán.

 Cancelar

Limpia la pantalla de datos sin registrar los datos ingresados.
