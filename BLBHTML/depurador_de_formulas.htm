Depurador de fórmulas

El depurador de fórmulas es una herramienta muy potente de BAS Laboro que permite recorrer el algoritmo de una fórmula para su prueba y control detallado.

Al crear una fórmula de cálculo, sobre todo si su algoritmo es extenso, resulta complejo verificar que dicha fórmula contemple todas las situaciones que puedan ocurrir.

El depurador permite asignar valores a variables y a acumuladores facilitando la tarea de verificación de resultados.

La asignación de valores puede efectuarse en forma manual, desde la tarea de definición del concepto, a través del generador de fórmulas, o en forma automática dentro de una liquidación, tareas que explicaremos en los párrafos siguientes.

Existen dos formas de acceder al depurador de fórmulas, una de ellas es desde el generador de fórmulas, en la tarea destinada a efectuar el alta de conceptos y la otra, durante el proceso de liquidación, efectuando la parametrización de dicha tarea.

En los siguientes párrafos se describen los elementos que forman parte del depurador y los procesos de depuración.

  • Elementos del depurador

Independientemente desde donde se acceda, la pantalla del depurador ofrece una serie de elementos que describiremos en los próximos párrafos. Dispone de un menú, "Variables", y de una barra de tareas para la depuración, ubicada en la parte superior de la pantalla. En la parte central se visualiza la fórmula a depurar y en la parte inferior se dispone de dos cuadros, uno que permite seleccionar los elementos a valorizar y el segundo que permite visualizar los valores de cada uno de ellos.

  • Menú Variables

Este menú dispone de las tareas "Cargar valores" y "Guardar valores" que permiten almacenar y tomar valores de variables y acumuladores utilizando archivos de textos.

  • Cargar valores

Esta opción permite importar desde un archivo de texto, con extensión "xml", valores de variables y acumuladores para poder depurar la fórmula de un concepto sin tener que cargar dichos valores en forma.

  • Guardar valores

Esta opción permite guardar en un archivo de texto, valores asignados a las variables de una determinada fórmula de un concepto, automáticamente durante una liquidación, o en forma manual en la depuración de un concepto, sin que éste esté vinculado a una determinada liquidación.

  • Barra de tareas

La barra de tareas está conformada por una serie de botones que permiten efectuar el control y seguimiento de las fórmulas de liquidación. Las tareas que efectúan cada uno de ellos se describe en los siguientes párrafos.

Continuar

Presionando este botón, el sistema recorre el algoritmo de la fórmula.

Alternar puntos de interrupción

Este botón establece puntos de detención en la ejecución del proceso de depuración. Su utilización permite obtener los valores asignados por el sistema a las diferentes variables, funciones y acumuladores vinculados con la fórmula depurada.

Paso a paso

Recorre las líneas de la fórmula una a una resolviendo los cálculos correspondientes y asignando valores a los elementos involucrados.

Finalizar depuración de fórmulas

Permite abandonar, cancelando, la depuración de una fórmula. Al finalizar ofrece como resultado el valor cero.

  • Depuración de fórmula en el concepto

Para depurar un concepto, se ingresa en la tarea que permite dar de alta los conceptos, se selecciona la clase y el concepto que se va a depurar, se presiona el botón "Modificar" y se ingresa al Generador de fórmulas presionado el botón correspondiente.

Para acceder a la tarea de depuración se presiona el botón "Depurar" que se encuentra en la parte derecha de la pantalla.

Una vez que se accede al depurador se deben valorizar las variables y acumuladores que intervienen en la fórmula, es decir, seleccionar las variables y acumuladores e informar los valores para cada uno de ellos.

Para asignar un valor, por ejemplo a una variable definida por el usuario, se selecciona la carpeta correspondiente en la parte inferior-izquierda de la pantalla y se selecciona el elemento a valorizar en la parte inferior-derecha de la pantalla de depuración. Se hace doble clic con el botón izquierdo del mouse sobre la variable y aparece una ventana emergente que permite el ingreso de un valor para asociar a la variable seleccionada.

Cuando la fórmula involucra acumuladores se deben asignar los valores en forma previa.

Una vez terminado este proceso es necesario establecer los puntos de detención en caso que interese observar la valorización de cada elemento de la fórmula y la metodología de depuración (Paso a paso o presionando el botón Continuar).

Para indicar puntos de detención se avanza con la barra de desplazamiento situada a la derecha de la pantalla del depurador hasta visualizar la línea de la fórmula donde queremos que se detenga la depuración.

Una vez situados en la línea se debe presionar el botón de detención, apareciendo un punto en el margen izquierdo indicando donde se detendrá el proceso.

Para depurar con un recorrido rápido se debe presionar el botón "Continuar" y el sistema recorrerá la fórmula deteniéndose en los puntos de detención, si existieran, o en los acumuladores, ya sea para que le ingresemos un valor o para ofrecer el informado previamente.

Para depurar línea por línea se debe presionar el botón "Paso a paso" en forma reiterada para avanzar en el recorrido. Esto permite observar cual es la secuencia de cálculo que efectúa el sistema dependiendo de los valores de los elementos que intervienen.

Esta última opción permite, por ejemplo, visualizar el valor asignado a una variable temporal dentro de la fórmula con solo posicionar el mouse sobre dicha.

  • Depuración de fórmula en la liquidación

Para depurar una fórmula de un concepto durante una liquidación, se ingresa en la tarea "Liquidaciones" del menú "Sueldos y Jornales".

Una vez en la tarea se debe seleccionar el/los legajos que se desean liquidar depurando sus conceptos, se tilda el check box "Depurar conceptos" y se presiona el botón "Aceptar" para iniciar el proceso.

El sistema ofrece una pantalla para parametrizar el proceso de depuración, es decir, para que se indique en que conceptos se detendrá la liquidación o se efectuará la depuración.

Los puntos de detención permiten, una vez interrumpida la liquidación, presionar el botón "Inspeccionar variables" y visualizar los importes correspondientes a los conceptos liquidados y, presionando el botón "Inspeccionar variables", consultar el valor que han asumido todas las variables relacionadas con el legajo liquidado.

Es posible eliminar puntos de detención presionando el botón "borrar puntos de interrupción" o, en caso de querer continuar con la liquidación, se deberá presionar el botón "Continuar".
